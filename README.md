### Node.js

    npm install array-enpansion -S

    import "array-enpansion"

## API

### max

```js
var arr = [1,2,3,4,34,54]
arr.max()
// 54
```

### min

```js
var arr = [1,2,3,4,34,54]
arr.min()
// 1
```

### unique

```js
let arr = [1,1,2,2,"2",undefined,undefined]
arr.unique()
// [1,2,"2",undefined]
```
